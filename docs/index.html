Qrgp - qr code generator primitive

Basic app to generate QR code for MRE platform mobile phone (including Nokia S30+). Tested on Nokia 225 with display resolution 240x320. If exists automatically open "newqr.txt" file on storage card or phone memory. RIGHT_SOFTKEY - exit app. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application file - "Qrgp.vxp".
