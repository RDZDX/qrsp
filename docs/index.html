Qrsp - qr code scanner primitive

Basic app to scann QR code for MRE platform mobile phone (including Nokia S30+). Tested on Nokia 225 with display resolution 240x320. Scanned text saved on "e:\Qrsp.txt" (changing app name change default text file name). Additionaly scanned text saved to SMS draft (if it is not greater 500 symbols). Second scan initialized with button LEFT_SOFTKEY save text to new file. Previous file renamed to autogenerated name from date and time. LEFT_SOFTKEY - next scann, RIGHT_SOFTKEY - exit app, KEY_DOWN - next page, KEY_UP - previous page. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application file - "Qrsp.vxp". (For low memory phone: Qrsp_850KB.vxp").
